Alt Framework is a JavaScript-based web application framework built on top of Java Servlets.

Alt Framework is changing! Read the new Alt Manifesto.

Alt Framework is currently in alpha development stages, but a working server is available for download through SVN.

Check out the Proposal and Foreword pages for a project overview. There is already a wealth of information in the documentation section.

Onion ML

Onion ML is an XML-based template system designed with a bias toward modularity and simplicity.

A rough comparison can be made to XSLT and JSF (Java Server Faces), Onion ML provides a means of defining custom XML tags in an attempt to abstract modular content design.

These custom tags can both be designed directly in Onion ML xml files, or they can be defined as a functional code which generates data on the fly.

In addition to static content, Onion ML provides simple yet powerful control flow methods necessary for dynamic content. Methods for iterating over data sets and conditionally displaying tags are core to Onion ML's functionality.

Contribute

The Alt Framework is an open source project following the unrestrictive zlib license (license information).

We decided to keep the source open to make it easy and free to setup while also allowing users to easily contribute features and modify the framework to their needs.

Subversion

Using your favorite SVN client (TortoiseSVN is a nice Windows client), download the source from:

svn://cellosoft.com/alt/trunk

This gives you access to the latest source (and last built binary) from our code repository.

Alt Framework alpha

This version is a (functional) developer snapshot of the Alt Framework. It includes both the binaries necessary to run an Alt Framework server as well as the source code.

It bundles everything you need, but should only be considered by people who want to get a taste for the Alt Framework:

  • AltServlet v0.05 - the
    • Rhino CVS (with custom patches) - JavaScript interpreter
    • XMLBeans's xbean.jar - needed for E4X support in Rhino
  • Alt Framework core libraries - the framework itself, written in JavaScript
  • AltGUI - a simple interactive GUI for running and monitoring a test server
  • Jetty 6 RC - an embedded Java application server
  • HSQL - an embedded relational database
  • Jakarta Commons FileUpload - a library for handling HTTP multipart encoded forms
  • MySQL Connector/J - provides MySQL connection support for Java

Coming soon!

You can also grab the latest "bleeding edge" version from SVN. Read the Contribute page.

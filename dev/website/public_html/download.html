RhinoServlet v0.02

The RhinoServlet is a Rhino-based JavaScript web application server. Inspired by Helma and Rhinola, RhinoServlet is a clean, simple, and modular JavaScript pusher for any Java Servlet system.

The standard download includes a copy of the Jetty6 beta and a HTTPServer class (and GUI, shown below) that makes it easy to jump in and get started!
